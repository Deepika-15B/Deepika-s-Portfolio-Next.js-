##  Setup Instructions

### Prerequisites
- Node.js (v16+)
- npm

---

### Installation

1. Clone the repository

git clone https://github.com/your-username/portfolio.git


2. Navigate to project

cd portfolio


3. Install dependencies

npm install


4. Run project

npm run dev


5. Open browser

http://localhost:3000

- Domain from Freenom

3. Deploy
4. Add custom domain
